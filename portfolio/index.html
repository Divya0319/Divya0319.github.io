RaahiReserve

A comprehensive Spring Boot application designed to simplify bus booking and route management for a seamless travel experience.

  • Core Functionality: Book trips from selected source and destination locations with ease, assign buses and seats based on passengers' preferences, and mark passengers as traveled on their scheduled date (travel simulation).
  • Payment Integration: In-house payment gateway with multiple modes for a smooth booking payment experience, featuring a timeout mechanism to enhance security.
  • Performance Optimization: Java multithreading is utilized to reduce server startup time.
  • Security: All endpoints are secured using Spring Security, preventing unauthorized access.
  • Entity Relationships: Database schema designed with thoughtful entity relationships and mappings.
  • User-Friendly Features: Easily view past booking details with just a few clicks.

Previously deployed on AWS Elastic Beanstalk with a MySQL database hosted on AWS RDS.
PayLinkSim

A RazorPay-integrated payment gateway simulation that securely generates payment links, tracks transactions, and manages statuses with JWT-authenticated APIs.

  • Core Functionality: Creates dummy orders with custom order amounts and interacts with the RazorPay API to generate payment links.
  • Transaction Handling: Consumes RazorPay responses (success/failure), updates payment statuses, and records the details in the database.
  • API Documentation & Testing: SpringDoc OpenAPI is integrated for seamless API testing and documentation.
  • Security: JWT token authentication secures all endpoints, with token validation logic ensuring the authenticity and validity of tokens.
  • Entity Relationships: Database schema designed with thoughtful entity relationships and mappings.
  • User-Friendly Features: Easily view past booking details with just a few clicks.

Previously deployed on AWS Elastic Beanstalk with a PostgreSQL database hosted on AWS RDS.
